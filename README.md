# Project Aegis

## Overview

Project Aegis is an AI-assisted Trust & Safety Investigation Framework designed to analyze large-scale user-generated content datasets and support abuse detection workflows.

The project combines data analysis, risk scoring, prompt engineering, and evaluation techniques to simulate real-world Trust & Safety operations similar to those used by content moderation and platform integrity teams.

## Dataset

Jigsaw Toxic Comment Classification Dataset

Size:

* 159,571 comments
* 6 abuse categories

Categories:

* Toxic
* Severe Toxic
* Obscene
* Threat
* Insult
* Identity Hate

## Features

### Data Ingestion

* Modular CSV loading
* Validation and logging

### Content Analysis

* Abuse category profiling
* Distribution analysis
* Statistical summaries

### Investigation Workflows

* High-risk comment identification
* Risk scoring framework
* Abuse prioritization

### Prompt Engineering

* Prompt Version 1
* Prompt Version 2

* Classification accuracy calculations
* Prompt performance benchmarking
